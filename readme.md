# InstantWeb LivePreviewer

Live preview your HTML, CSS, and JavaScript files directly inside ACode — powered by the InstantWeb local server app. No Termux, no Node.js, no PC required.

Super lightweight setup — InstantWeb app is only **3.9MB** to download and **~11MB** installed. No extra dependencies, no complex configuration.

[<img src="https://img.shields.io/badge/Open%20Source-GitHub-181717?logo=github&style=for-the-badge" />](https://github.com/TechSetuApps/instantweb-livepreviewer)

---

## Requirement

This plugin requires the **InstantWeb** app to function. InstantWeb runs a local HTTP server on your Android device that serves your HTML files.

Download InstantWeb:
[APKPure — InstantWeb](https://apkpure.net/instantweb/app.techsetuapps.instantweb)

---

## Installation Steps

**Step 1 — Set Up InstantWeb**

1. Download and install InstantWeb from the link above.
2. Open InstantWeb.
3. Tap **Select File** and choose any `.html` file from your device.
4. Tap **Start Hosting** — the server starts on `http://127.0.0.1:8080`.

**Step 2 — Install This Plugin**

1. Open ACode.
2. Tap the menu icon in the top-left corner.
3. Tap **Plugins**.
4. Search for **InstantWeb LivePreviewer**.
5. Tap **Install**.

---

## How to Use

1. Open ACode and open any `.html` file.
2. Make sure InstantWeb is running and hosting your file.
3. Tap the floating **W** button on screen, or press **Ctrl + Shift + W**.
4. The live preview panel opens showing your page in real time.
5. Save your file — preview auto-reloads within 500ms.

---

## Features

- Live preview with 500ms auto-reload
- Portrait and landscape layout support
- Resizable preview panel
- Fullscreen preview mode
- Draggable floating W button
- Built-in Eruda JS console (available in fullscreen mode)
- Keyboard shortcut: Ctrl + Shift + W
- No Termux, no Node.js, no PC required
